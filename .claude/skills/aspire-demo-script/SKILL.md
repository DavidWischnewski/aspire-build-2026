# /aspire-demo-script

Run at minute 110. Takes 5 minutes. Do not wing the demo.

## Input

Ask:
1. What did you actually build? (one sentence — the real thing, not the original brief)
2. Is it deployed? (URL, or local only)
3. Who in the room do you most want to impress?

## Draft the 30s script

**[0:00–0:05] Hook**
"I built [thing] because [specific frustration]."
Make it personal. Not "our team needed a tool." Make it: "I spend 45 minutes every Monday doing X."

**[0:05–0:15] The demo**
3 clicks or less. Show the moment of value — not the login, not the settings, the thing that makes it worth watching.
Pre-load real-looking data. No spinners during demo if possible.

**[0:15–0:25] The wow**
The one thing that's only possible because of Claude. What would break if you removed the AI?

**[0:25–0:30] The kicker**
One sentence: what's the next iteration. One specific thing, not "we could do so much more."

## Coaching notes

- Say the hook with confidence — it's the only line that must land
- Don't explain the tech stack. Nobody cares.
- Pause after the wow. Let it breathe.

## Cheat-sheet

Output a 5-bullet version for a second monitor:
1. Hook (memorize this)
2. Click 1
3. Click 2 → wow
4. The wow line (exact words)
5. Kicker (exact words)
